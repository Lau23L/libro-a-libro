# MVP – Libro a Libro

El MVP (Minimum Viable Product) será una versión simple del
producto que permita validar la idea.

## Funciones principales

### Usuarios
- Registro
- Inicio de sesión
- Perfil básico

### Libros
- Publicar libro
- Fotos del libro
- Estado del libro
- Precio o intercambio

### Búsqueda
- Buscar por título
- Buscar por autor

### Mensajes
- Contactar al dueño del libro
